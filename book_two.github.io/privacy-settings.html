EmailJS Setup Guide

To enable newsletter subscriptions, follow these steps:

Step 1: Create EmailJS Account

  1. Go to emailjs.com
  2. Sign up for a free account
  3. Verify your email

Step 2: Connect Email Service

  1. In EmailJS dashboard, go to "Email Services"
  2. Click "Add Service" and select "Gmail" (or your preferred service)
  3. Follow the authentication steps
  4. Copy your Service ID

Step 3: Create Email Template

  1. Go to "Email Templates"
  2. Click "Create New Template"
  3. Name it: flipbook_newsletter
  4. Use this template structure: 
    To: {{to_email}}
Subject: {{subject}}

{{message}}

---
FlipBook Pro Newsletter
garrickkogola@gmail.com
  5. Click "Save" and copy the Template ID

Step 4: Get Your Public Key

  1. In dashboard, click your profile → "Account"
  2. Find "Public Key" section
  3. Copy your Public Key

Step 5: Add Credentials to Code

In your JavaScript file, replace the placeholder values:

// BEFORE:
emailjs.init("YOUR_PUBLIC_KEY");

// AFTER:
emailjs.init("your_actual_public_key_here");

// And in email send:
await emailjs.send('YOUR_SERVICE_ID', 'flipbook_newsletter', { ... });
// BECOMES:
await emailjs.send('your_service_id', 'flipbook_newsletter', { ... });
Free Tier: EmailJS offers 200 emails/month free. Perfect for newsletters!

Alternative: Manual Newsletter

If you don't want to set up EmailJS, users can subscribe manually:

  1. Collect emails in a spreadsheet
  2. Use a service like Mailchimp (free tier available)
  3. Send newsletters through your email provider
